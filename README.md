# Smart Traffic Light System Using IoT

## Repository Maintainer
**Sunhera Fathima**

## Project Overview
This repository contains the project documentation, presentation, source code, and publication related to the Smart Traffic Light System Using IoT. The project aims to reduce traffic congestion by dynamically controlling traffic signals based on real-time vehicle density using IoT technologies.

## Features
- Intelligent traffic signal control
- Real-time vehicle detection
- IoT-based monitoring
- Reduced congestion and waiting time
- Scalable for smart city applications

## Hardware
- ESP32/NodeMCU
- IR/Ultrasonic Sensors
- Traffic Signal LEDs
- Breadboard
- Jumper Wires
